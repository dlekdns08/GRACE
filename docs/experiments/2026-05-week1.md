### API surface verified

| Aspect | What we checked | Result |
|---|---|---|
| Layouts | `OvercookedGridworld.from_layout_name('cramped_room' / 'asymmetric_advantages')` | OK |
| Action space | `len(Action.ALL_ACTIONS)` | 6 (N, S, E, W, STAY, INTERACT) — matches config |
| Featurization | `OvercookedEnv.featurize_state_mdp(state)` | per-agent 96-D float vector — matches `env.obs_dim=96` |
| State text | `state.players[i].held_object`, `state.has_object(loc)`, `mdp.get_pot_locations()`, `SoupState.{is_ready, is_cooking, ingredients, cook_time_remaining}` | all present; wired into `_carroll_state_to_text` in `src/envs/python_env.py` |
| 100-step rollout | random actions, `cramped_room` layout via `PythonOvercookedEnv` | no exceptions |

The MLAM (medium-level action manager) is built lazily on first `featurize_state_mdp` call (~10ms cached on disk under the package's `data/planners`). Subsequent constructions hit the disk cache.

### Code changes

- `src/envs/python_env.py`: switched from the spatial `lossless_state_encoding` (520-D after flatten) to `featurize_state_mdp` (96-D) so observations match the config; replaced the placeholder text path with `_carroll_state_to_text` that mirrors the deterministic format used by the dummy env.
- `scripts/train.py`: final + periodic (`every eval_every` env steps) checkpoints, post-training eval that writes `eval_results.parquet`, surface `n_invalid_subgoals` in iteration logs.
- `scripts/eval.py`: tolerates the new metadata-wrapped checkpoint dict; downgrades `load_state_dict` mismatches to a warning.
- `src/llm/parsers.py`: new `parse_subgoal_with_validation` (strict, opt-in). Original `parse_subgoal` unchanged so existing callers aren't disturbed.
- `src/training/rollout.py`: per-agent enum validation drops invalid subgoals to `None` rather than aborting the rollout; `RolloutBatch.n_invalid_subgoals` exposes the count for logging.
- New tests: `tests/test_subgoal_validity.py` (9 cases), `tests/test_checkpoint.py` (3 cases — full subprocess train -> load -> eval round trip).
